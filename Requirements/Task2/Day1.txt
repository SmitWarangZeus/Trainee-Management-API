Task 2.1: Configure MySQL Provider 
    • Replace UseInMemoryDatabase with MySQL database configuration. 
    • Read the database connection string from appsettings.json. 
    • Do not hardcode connection strings inside controllers or services. 
{ 
    "ConnectionStrings": { 
        "DefaultConnection":  
            "server=localhost;port=3306;database=trainee_management_db;user=root;password=your_password;" 
    } 
}
